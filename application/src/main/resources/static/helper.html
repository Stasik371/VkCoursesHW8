Справка по работе с веб-сервисом на основе Jetty

Роли:

    1. Guest, доступен GET запрос. Логин/Пароль guest/guest
    2. Manager, доступны GET и POST запросы. Логин/Пароль manager/manager

Запросы:

    1. GET http://localhost:PORT/prod/allProducts отправка GET запроса(получение json строки всех позиций).
    2. POST http://localhost:PORT/prod/addProduct/INFO отправка POST запроса, где INFO: Название позиции:Название организации:Количество.
    3. GET http://localhost:PORT/prod/ProductsWithRightOrganization/ORGANIZATION отправка POST запроса, где ORGANIZATION: Название организации
    4. POST http://localhost:PORT/prod/deleteProduct/NAME отправка POST запроса, где NAME: Наименование продукта
